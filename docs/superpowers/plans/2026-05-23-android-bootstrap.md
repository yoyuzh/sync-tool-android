# Android Bootstrap Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first native Android repository for sync-tool with Kotlin, Jetpack Compose, AccessibilityService, InputMethodService, WorkManager, Room, and clean domain/data boundaries.

**Architecture:** The app is a single native Android application module with layered packages under `com.yoyuzh.cliplink`. UI, IME, Accessibility, workers, notifications, and SMS placeholders depend on domain use cases; data implements repository interfaces with Room and a remote API client. Accessibility is the realtime event entry, IME is the insertion surface, and WorkManager owns low-frequency sync and retry.

**Tech Stack:** Kotlin, Android Gradle Plugin, Gradle Wrapper, Jetpack Compose, Hilt, Room, WorkManager, Retrofit or Ktor Client, kotlinx.serialization, Kotlin Coroutines and Flow, AndroidX Test
